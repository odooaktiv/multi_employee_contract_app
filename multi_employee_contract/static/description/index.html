How it Works

Select any one or more employee


Click on Action, Click "Create Contract"


Added Wages and Salary Structure of the Employee


After click Confirm all selected employee's Contract will be created.


If Contract is already exist and contract's state is New,Open, or pending so, it's show warning and it's not allows to create new contract.



If Contract is already exist and contract's state is Closed that can allow to create new contract and re-direct on contract.
