Multiple Employee Contract

Odoo’s default functionality allowed only single employee contract to be created at a time. But we have made an effort to reduce the hassles of creating contract for every employee separately. With this module, user can select multiple employees and create contract for them by specifying their salary and contract type.

Moreover, if same contract exists for employee, the module will restrict from creating/duplicating same contacts. If any employee’s contract is expired and user tries to create similar contract then module will allow to create the contract.

Prerequisites before Installation

No prerequisites needed.

Configurations

No configurations.

You can select multiple employees for creating their respective contracts.

On clicking the Action button, you will be displayed with a “Create Contract” option. So, click on “Create Contract” to create contract for the selected employees.

After having selected the employee contracts to be created, you will be displayed with a wizard wherein you can fill up the “Wage” and “Salary Structure” for each employee separately.

By confirming the contract creation for selected employees, the newly created contracts will be redirected on “Contracts” page.

If the contract start and end dates are same of same employee, then that contract will not be created.

It will pop up a warning saying- “Contract already exists”.

If the contract which already exists but the contract status is expired then the module will allow to create the contract with same start and end date for same employee.
